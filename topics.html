Course material by topic¶

This is an arrangement of the material for the course by topic, rather than by teaching day.

Python¶

  • Brisk introduction to Python. See August 29: introduction, Python, images;
  • “for” and “while”, “break” and “else:”;
  • Modules and scripts;
  • Packages and namespaces;
  • List comprehensions;
  • Making and breaking file paths in Python. See: Git and github workflow;
  • Where does Python look for modules?. See: Git and github workflow;
  • Two double underscore variables;
  • Using PYTHONPATH.

Numpy, arrays and images¶

  • What is an image?. See August 29: introduction, Python, images;
  • NumPy introduction (from scipy lecture notes (SLN);
  • numpy array object (SLN);
  • array operations (SLN). See: Lab 01 exercise;
  • Array reduction operations;
  • Arrays as images, images as arrays. See: September 12: arrays, images and plotting;
  • Reshaping and three-dimensional arrays. See: September 12: arrays, images and plotting;
  • Index ordering and reshape in NumPy and MATLAB;
  • Working with four dimensional images, masks and functions. See: September 19: version control, 4D arrays, diagnostics;
  • Reshaping, 4D to 2D;
  • Logical operations on boolean arrays;
  • Reshaping 4D images to 2D arrays;
  • Slicing with boolean vectors. See: October 3: statistics on brain images;
  • Indexing with boolean masks.
  • Vector and matrix dot products, “np.outer”;
  • Testing for near equality with “allclose”;
  • Numpy arange;
  • Methods vs functions in NumPy;
  • Subtracting the mean from columns or rows;
  • Adding length 1 dimensions with newaxis

Matplotlib¶

  • Plotting lines in matplotlib;
  • Subplots and axes in matplotlib.

Statistics and brain images¶
