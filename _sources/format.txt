###################
Format of the class
###################

* Prior reading / homework for each week of about 30 minutes;

* Class is 2 hours:

    * 10 minutes debrief from previous class
    * 40 minutes talk introduction + questions;
    * 60 minutes problems;

As we cover the material, we will also pause at times to cover how to do:

* version control;
* code collaboration.
